## Anti Desk Time ##

## Anti Desk Time is the solution built for developers who are tired filling the gap between their desk time.

## How to run? ##

# Step 1: Make sure the python installed on your system along with pip
# Step 2: Create virtual environment `py -m venv venv`.
    # Note: This will create a seperate virtual environment. So, it will not effect your other python projects.
# Step 3: Activate virtual environment
    # For Windows Users `venv\Scripts\activate`.
    # For Linux Users `source venv\bin\activate`.
# Step 4: Run the command `pip install -r requirements.txt` to install the required package. Here, it is pyautogui.
# Step 5: Run following command to execute code `py index.py`.
# Step 6: Voila! Now, your mouse cursore starts moving which will get recorded in the desktime resulting in no gaps.
# Step 7: To stop the script Press `Ctrl + C` in terminal.
